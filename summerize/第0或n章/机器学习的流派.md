# 机器学习的流派、人工智能主义及数学支撑

根据图中内容，机器学习主要方法、对应的人工智能主义流派及其数学支撑如下：

| 方法         | 人工智能主义流派 | 数学支撑                   |
|--------------|------------------|----------------------------|
| 决策树（DT） | 符号主义         | 信息论、概率论、统计学     |
| 规则/逻辑    | 符号主义         | 逻辑学、集合论             |
| 神经网络（NN）| 连接主义         | 线性代数、微积分、概率论、优化理论 |
| 支持向量机（SVM）| 连接主义/符号主义 | 凸优化、核方法、线性代数、统计学习理论 |
| 贝叶斯方法    | 贝叶斯主义       | 概率论、贝叶斯定理、统计推断 |

- 符号主义：强调知识的符号表达与推理。
- 连接主义：强调模拟人脑神经元连接结构。
- 贝叶斯主义：强调概率推断和不确定性建模。

这些方法在实际应用中常常结合使用，数学支撑也在不断发展。
